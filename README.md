# eggApi


- 本项目打算在egg.js 基础模板上添加常用的项目中的功能， 比如
  + 单点登录 sso
  + 跨域资源共享 cros
  + 文件上传，断点续传
  + 文件下载
  + api请求转发
  + excel 文件插入到数据库， 将数据库数据导出到excel
  + n级联动数据格式读取
  +
